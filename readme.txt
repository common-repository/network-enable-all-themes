It's frustrating to have to enable themes when using Multisite as a local development environment. This plugin keeps all of them activated.

== Description ==

I already described it above. Use it in a development environment only!

== Frequently Asked Questions ==

= Why don't I see the plugin? =

It's a network-only plugin. Go to the plugins screen in the network admin and enable it there.
